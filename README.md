# 股票筛选器

A 股个人选股工具，支持按基本面和技术面指标多条件筛选。

## 快速开始

### 后端
```bash
cd backend
pip install -r ../requirements.txt
uvicorn main:app --reload    # API 运行在 http://localhost:8000
```
首次启动时自动从新浪财经拉取数据（约 60 秒），也可手动执行 `python seed_data.py`。

### 前端
```bash
cd frontend
npm install
npm run dev                  # 界面运行在 http://localhost:5173
```

## 技术栈
- 后端：Python 3.11+, FastAPI, SQLAlchemy, pandas, akshare (K线), 新浪财经 API (选股指标)
- 前端：Vue 3 (Composition API), TypeScript, Vite
- 数据库：SQLite

## 功能

### 股票筛选
- 多条件筛选：PE (TTM)、PB、ROE、市值、股息率、营收增长率、换手率、涨跌幅、量比
- 按市场板块过滤（沪深A股、创业板、科创板、北交所）
- 概念板块过滤（支持搜索和复选）
- 关键字搜索（股票名称或代码）
- 排除 ST 股票
- 自选股过滤（仅看自选）
- 表格排序（点击标题：升序 → 降序 → 取消）和分页
- CSV 导出

### 股票详情（侧边抽屉）
- 基本信息展示（代码、名称、市场、行业、上市日期）
- 分时图（当日价格线 + 均价线 + 成交量，含昨收线）
- K 线蜡烛图（日K / 周K / 月K 切换，含 MA5/MA20/MA60 均线 + 成交量柱）
- MACD 指标图（DIF/DEA/MACD 柱）
- 最近 120 个交易日的 OHLCV 数据表

### 策略管理
- 6 个内置策略模板（高ROE成长股、低估值蓝筹、均线多头排列、新高附近、放量突破、低位高换手）
- 自定义策略：保存/加载/删除（内置策略不可删除）
- 快捷策略栏（股票筛选页顶部标签）
- 策略仪表盘：多策略运行概览（匹配数 + Top 5 股票）、策略交集分析

### 板块排名
- 行业板块涨跌幅排名
- 展示换手率、领涨股票及涨幅

### 涨跌停板
- 涨停板统计（连板数、封单量、封板时间、行业）
- 跌停板统计
- 点击股票弹出详情

### 其他
- 一键刷新数据（从新浪财经拉取全量数据）
- 明暗主题切换
- 响应式布局
